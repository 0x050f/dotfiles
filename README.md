# .vimrc

My simple vimrc config and auto installer

## Installation 

```sh
git clone https://github.com/ska42/myvimrc /tmp/myvimrc
sh /tmp/myvimrc
```

## Mapping

| Key | Map | Comment |
| --- | --- | ------- | 
| j | gj | visual movement down |
| k | gk | visual movement up |
| \<space> | za | open fold |
