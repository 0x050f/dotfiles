# .vimrc

.vimrc & plugins install

## Installation 

```sh
git clone https://github.com/SimplySka/.vimrc /tmp/.vimrc
source /tmp/.vimrc/install.sh
```

## Mapping

| Key | Map | Comment |
| --- | --- | ------- | 
| j | gj | visual movement down |
| k | gk | visual movement up |
| B | ^ | begin of the line |
| E | $ | end of the line |
| \<space> | za | open fold |
