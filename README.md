```sh
source /tmp/.vimrc/install.sh
```

## Mapping

| Key | Map | Comment |
| --- | --- | ------- | 
| j | gj | visual movement down |
| k | gk | visual movement up |
| B | ^ | begin of the line |
| E | $ | end of the line |
| \<A-t> | :Le | :Lexplore to open/close files tree |
| \<space> | za | open fold |
